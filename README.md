### Getting Started
1. Clone the project 
```
git clone https://github.com/tsa87/Intuitive-Gesture-Drone-Control.git
```
2. Install the prerequisites
```
pip install Pillow
pip install pandas
pip install keras
pip install sklearn
pip install opencv-contrib-python
pip install matplotlib
```
3. Train on the dataset
```
python train.py

```
4. Run the demo
```
# please make sure 
python test.py
```
